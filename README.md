# University of San Carlos - NetSuite Customizations

This repository contains NetSuite SuiteScript customizations for the University of San Carlos.

## Structure

```
Sandbox/
├── Prints/
│   ├── Advanced pdf prints/    # PDF templates for transaction printing
│   └── Suitelet Prints/        # Suitelet-based print solutions
└── Scripts/
    └── Client Scripts          # Client-side validation scripts
```

## Contents

### Scripts
- **Cost Center Validation** - Client script for validating cost centers on transactions

### Prints
- **Check Voucher** - Suitelet for printing check vouchers
- **Journal Voucher** - Suitelet for printing journal vouchers
- **Advanced PDF Templates** - Custom PDF templates for transaction output

## Development

Scripts follow the SuiteScript 2.x API standard.
